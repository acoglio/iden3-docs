go-iden3
========

Go implementation of the iden3 system.

`Go Report
Card <https://goreportcard.com/report/github.com/iden3/go-iden3>`__
`Build Status <https://travis-ci.org/iden3/go-iden3>`__

Install
-------

::

   $ go get github.com/iden3/go-iden3
